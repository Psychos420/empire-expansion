# Lead-Generation-Funnel: Finanzielle Freiheit / Vermögensaufbau / FIRE

**Ziel:** Systematischer, automatisierter Funnel zur Gewinnung qualifizierter Leads 35+ und Conversion in kostenpflichtige Produkte.

---

## Funnel-Übersicht

```
Awareness → Lead Magnet → Nurture → Tripwire → Core Offer → Profit Maximizer
   │            │            │          │            │              │
 YouTube    FIRE-Schnell-  E-Mail-   97-EUR      497-EUR      997-EUR Premium/
 LinkedIn   check / PDF    Sequence  ETF-Kurs    FIRE-Kurs    Coaching / Club
 SEO        (Free)                    (Low)       (Mid)        (High/Recurring)
```

---

## Phase 1: Awareness (Bekanntschaft)

**Ziel:** Reichweite aufbauen und Vertrauen durch wertvollen Content schaffen.

### Kanäle und Inhalte

| Kanal | Content-Beispiel | CTA |
|---|---|---|
| YouTube | „FIRE mit 40? Realitätscheck für 35+" | Link zu Lead Magnet in Beschreibung |
| LinkedIn | „3 Steuertricks, die dein Bankberater verschweigt" | Link zu Blog/Lead Magnet |
| SEO-Blog | „Sparer-Pauschbetrag 2026: So nutzt du ihn richtig" | Inline-CTA für Lead Magnet |
| Podcast-Gast | Diskussion über Coast FIRE | Nennung der Website |
| Kurzvideo | „Dieser ETF-Fehler kostet dich 50.000 EUR" | Link im Profil |

### Tools
- **YouTube:** Kanal + Thumbnails (Canva)
- **LinkedIn:** Persönliches Profil + Unternehmensseite
- **Website:** WordPress / Webflow / Ghost
- **SEO:** Ahrefs oder Ubersuggest für Keyword-Recherche

---

## Phase 2: Lead Magnet (Erste Kontaktaufnahme)

**Ziel:** E-Mail-Adresse im Austausch für sofortigen Wert erhalten.

### Primärer Lead Magnet
**„Der FIRE-Schnellcheck für 35+"**
- 10 Fragen zur aktuellen finanziellen Situation
- Individuelle Auswertung als PDF
- Konkrete nächste 3 Schritte
- Geschätzter FIRE-Zeitpunkt

### Sekundäre Lead Magneten
1. **„Die ETF-Einsteiger-Checkliste"** (PDF)
2. **„Steuer-Checkliste für Anleger 2026"** (PDF)
3. **„7-Tage-E-Mail-Kurs: In 7 Tagen zum ersten ETF-Sparplan"**
4. **„FIRE-Rechner"** (Interaktives Tool / Google Sheet)

### Umsetzung
- Landing Page mit klarem Headline, Vorteilen und Opt-in-Formular
- Doppelte Opt-in-Pflicht für DSGVO-Konformität
- Sofortiger PDF-Download nach Bestätigung
- Segmentierung nach Lead-Magnet (z. B. ETF vs. Steuern vs. FIRE)

### Tools
- **Landing Page:** Webflow / Carrd / WordPress + Elementor
- **Formular:** ConvertKit / Mailerlite / Brevo
- **Zahlung:** Stripe (für spätere Phasen)

## Phase 3: Nurture (Vertrauensaufbau)

**Ziel:** Lead von „interessiert" zu „bereit zu kaufen" führen.

### Welcome-Sequence (7 E-Mails)

| Tag | Thema | Ziel |
|---|---|---|
| 0 | Willkommen + Lead Magnet | Erwartungen setzen, Branding |
| 1 | „Der größte Fehler bei der Geldanlage" | Problem aufzeigen |
| 2 | „Warum 35+ einen anderen Plan brauchen als 25-Jährige" | Zielgruppen-Fokus |
| 3 | Fallstudie: „Wie Anna mit 45 ihre Sparrate verdoppelte" | Sozialer Beweis |
| 4 | „3 ETF-Strategien, die wirklich funktionieren" | Wert liefern |
| 5 | „Was kostet dich dein Bankberater wirklich?" | Schmerzpunkt |
| 6 | Einladung zum Webinar / Einführung in Kurs | Soft Sell |
| 7 | Angebot: ETF-Einsteigerkurs für 97 EUR | Hard Sell |

### Segmentierungs-Fragen
Beim Opt-in oder in E-Mail 2:
- Alter (35–44 / 45–54 / 55+)
- Familienstand (Single / Verheiratet / Kinder)
- Beruf (Angestellt / Selbstständig / Beamter)
- Hauptziel (FIRE / Altersvorsorge / Steuern / Familie)
- Erfahrung (Anfänger / Fortgeschritten)

### Tools
- **E-Mail-Automation:** ConvertKit / ActiveCampaign / Brevo
- **Segmentierung:** Tags und Custom Fields in E-Mail-Tool
- **CRM:** HubSpot (kostenlos) oder Notion-CRM

## Phase 4: Tripwire (Erster Kauf)

**Ziel:** Erste Zahlung erhalten und Kaufschwelle senken.

### Produkt
**„Der ETF-Einsteigerkurs: In 21 Tagen zum ersten Sparplan"**
- Preis: **97 EUR**
- Umfang: 5 Module, 20 Videos, Downloads, Quiz
- Geld-zurück-Garantie: 14 Tage

### Verkaufs-Sequence (nach Tripwire-Kauf)
1. Danke-E-Mail + Zugangsdaten
2. Onboarding-E-Mail: „So holst du das Maximum raus"
3. Erfolgs-E-Mail nach Modul 3: „Du hast den schwierigsten Teil geschafft"
4. Upsell-E-Mail nach Abschluss: „Bereit für den nächsten Schritt?"

### Tools
- **Kursplattform:** Teachable / Thinkific / Kajabi / Podia
- **Zahlung:** Stripe / PayPal
- **Upsell:** Order Bump im Checkout

## Compliance-Hinweise im Funnel

- **DSGVO:** Doppeltes Opt-in, Datenschutzerklärung, Widerrufsbelehrung.
- **Affiliate-Kennzeichnung:** Jeder Affiliate-Link klar als Werbung kennzeichnen.
- **Haftungsausschluss:** Keine Anlageberatung, nur Bildung.
- **BaFin:** Keine Einzelwertempfehlungen, keine personalisierte Anlageberatung ohne Lizenz.
- **Widerrufsrecht:** 14 Tage für digitale Produkte (mit ausdrücklicher Zustimmung zur sofortigen Leistung).
