## Vernetzung der Bereiche (Ökosystem-Logik)

```
Traffic-Quellen (TikTok, SEO, Paid, LinkedIn, App Store)
         │
         ▼
   Lead-Magnet / Low-Ticket (PDFs, TikTok-Shop, App-Download)
         │
         ▼
   E-Mail / SMS / WhatsApp (Automatisiertes Nurturing)
         │
         ▼
   ┌─────────────┬─────────────┬─────────────┬─────────────┐
   │  Affiliate  │   PKV-Leads │  KI-Agentur │  App-Sub    │
   │   Sales     │  Vermittler │   Projekte  │ scriptions │
   └─────────────┴─────────────┴─────────────┴─────────────┘
```

Ein Lead aus dem PDF-Bereich kann gleichzeitig Interessent für PKV, Affiliate-Produkte oder die KI-Agentur sein. Jeder Bereich füttert die anderen.
