# LEAD-GENERATION-FUNNEL.md — TikTok Shop → Lead → Sale

> Schritt-für-Schritt-Funnel: Traffic-Quelle → Lead-Magnet/Angebot → Landing Page → E-Mail/SMS/WhatsApp-Follow-up → Sale/Termin. Inklusive Tools und Automatisierung pro Stufe.

---

## 1. Funnel-Überblick

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  Traffic-Quelle │────▶│  Lead-Magnet /   │────▶│  Landing Page   │
│  (TikTok etc.)  │     │  Einstiegsangebot│     │  (Conversion)   │
└─────────────────┘     └──────────────────┘     └────────┬────────┘
                                                          │
┌─────────────────┐     ┌──────────────────┐             │
│  Sale / Termin  │◀────│  Follow-up       │◀────────────┘
│  (High-Ticket)  │     │  E-Mail/SMS/WA   │
└─────────────────┘     └──────────────────┘
```

**Ziel:** Jede TikTok-Interaktion führt langfristig zu einem kontrollierbaren Kontakt (E-Mail/Telefon), der für hochpreisige Empire-Angebote qualifiziert werden kann.

## 2. Stufe 1: Traffic-Quelle

### 2.1 Primärquelle: TikTok

| Format | Zweck | Lead-Qualität |
|--------|-------|---------------|
| **Organische Videos** | Reichweite, Vertrauen, Follower | Mittel |
| **Shoppable Videos** | Direktverkauf + Profilbesuche | Hoch |
| **Spark Ads** | Skalierung erfolgreicher Videos | Hoch |
| **TikTok Lead Ads** | Direkte Lead-Erfassung | Sehr hoch |
| **Creator-Affiliate** | Reichweite + Social Proof | Mittel-Hoch |
| **Bio-Link / Profil** | Zentraler Umleitungspunkt | Hoch |

### 2.2 Traffic aus anderen Quellen (Synergien)

| Quelle | Content | Zweck |
|--------|---------|-------|
| **Instagram Reels** | Cross-Post TikTok-Videos | Zusätzliche Reichweite |
| **YouTube Shorts** | Cross-Post | SEO, längere Halbwertszeit |
| **Pinterest** | Video-Pins, Produkt-Pins | Inspiration, langer Tail |
| **Facebook-Gruppen** | Geteilter Content | Nischen-Community |
| **SEO-Blog** | Artikel zu „Besten Produkten ab 35" | Organischer Suchverkehr |

### 2.3 Traffic-Ziele pro Funnel-Typ

| Funnel-Typ | Hauptziel | Sekundärziel |
|------------|-----------|--------------|
| **Produktfunnel** | Direktverkauf im TikTok Shop | E-Mail-Sammlung via Produkt-Insert |
| **Lead-Funnel** | E-Mail-Adresse sammeln | Termin / Beratung vereinbaren |
| **Hybrid-Funnel** | Günstigen Einstieg verkaufen | Upsell / Cross-Sell via E-Mail |

| Produkt | Preis | Zweck |

## 4. Stufe 3: Landing Page

### 4.1 Struktur einer High-Converting Landing Page

| Abschnitt | Inhalt | Ziel |
|-----------|--------|------|
| **Hero** | Headline, Subheadline, Hauptbild/Video, CTA-Button | Aufmerksamkeit, erste Conversion |
| **Problem** | „Kennst du das..." — Pain Point adressieren | Identifikation |
| **Lösung** | Lead-Magnet / Produkt als Lösung präsentieren | Interesse |
| **Beweis** | Testimonials, Before/After, Statistiken, Logos | Vertrauen |
| **Nutzen** | 3–5 konkrete Vorteile | Abwägung |
| **Angebot** | Was genau bekommt der Nutzer? | Klarheit |
| **CTA** | Formular / Button wiederholt | Conversion |
| **FAQ** | Einwände vorwegnehmen | Bedenken abbauen |
| **Footer** | Impressum, Datenschutz | Rechtssicherheit |

### 4.2 Landing-Page-Typen

| Typ | Verwendung | Tool-Empfehlung |
|-----|------------|-----------------|
| **Squeeze Page** | Nur E-Mail sammeln | Carrd, ConvertKit, Leadpages |
| **Sales Page** | Direktverkauf Produkt | Shopify, Unbounce, Webflow |
| **Webinar-Registrierung** | Termin / Beratung | WebinarJam, Livestorm, Calendly |
| **Quiz-Funnel** | Segmentierung | Typeform, ScoreApp, Outgrow |

### 4.3 Conversion-Optimierung

- **Ladezeit <2 Sekunden**
- **Mobil-first** (90 % Traffic von TikTok ist mobil)
- **Eine einzige Aktion** pro Seite
- **Social Proof oberhalb der Falte**
- **Mehrfache CTAs**, aber identisches Ziel
- **Exit-Intent-Popup** mit Zweitangebot
- **TikTok Pixel** installiert für Retargeting

## 5. Stufe 4: E-Mail/SMS/WhatsApp-Follow-up

### 5.1 E-Mail-Sequenz nach Lead-Magnet (14 Tage)

| Tag | Betreff | Inhalt | Ziel |
|-----|---------|--------|------|
| 0 | „Hier ist dein Guide — und ein Geschenk" | Lead-Magnet + Begrüßung | Lieferung, Vertrauen |
| 1 | „Der größte Fehler ab 40" | Nützlicher Tipp + Story | Engagement |
| 2 | „Das habe ich selbst getestet" | Produkt-Empfehlung | Erste Soft-Sell |
| 4 | „3 Fragen, die ich ständig höre" | FAQ-Content | Einwände abbauen |
| 7 | „Nur für dich: 15 % Rabatt" | Angebot | Conversion |
| 10 | „Warum das funktioniert" | Wissenschaft/Erklärung | Vertrauen |
| 14 | „Letzte Chance: Rabatt läuft ab" | Urgency | Conversion |

### 5.2 SMS/WhatsApp-Sequenz

| Tag | Nachricht | Ziel |
|-----|-----------|------|
| 0 | „Danke! Hier ist dein Download: [Link]" | Bestätigung |
| 3 | „Hast du schon [Produkt] gesehen? 15 % für dich: CODE" | Conversion |
| 7 | „Frage zu [Thema]? Antworte hier — ich antworte persönlich." | Dialog |
| 14 | „Nur heute: Bundle-Deal" | Urgency |

### 5.3 Post-Purchase-Follow-up (nach TikTok-Shop-Kauf)

| Zeitpunkt | Kanal | Inhalt | Ziel |
|-----------|-------|--------|------|
| Sofort | E-Mail | Bestellbestätigung + Nutzungstipps | Onboarding |
| Tag 3 | E-Mail | „So bekommst du das beste Ergebnis" | Erfolg sicherstellen |
| Tag 7 | E-Mail | Bewertungsbitte + UGC-Aufforderung | Social Proof |
| Tag 14 | E-Mail | Cross-Sell / Refill | Wiederkauf |
| Tag 30 | E-Mail/SMS | Abo-Angebot oder Upgrade | CLV steigern |

## 6. Stufe 5: Sale / Termin

### 6.1 Direktverkauf (niedrigpreisig)

- TikTok Shop Checkout
- Shopify-Checkout (bei Bio-Link-Traffic)
- Ein-Klick-Upsells nach Kauf

### 6.2 Hochpreisige Angebote (Lead → Sale)

| Empire-Bereich | Angebot | Preis | Conversion-Pfad |
|----------------|---------|-------|-----------------|
| **PKV / Gesundheit** | Strategiegespräch | €0 | E-Mail → Quiz → Termin |
| **Finanzberatung** | Vorsorge-Check | €0–€49 | Lead-Magnet → Webinar → Termin |
| **Coaching** | Kennenlerngespräch | €0 | E-Mail-Sequenz → Calendly |
| **Agency** | Audit / Beratung | €0–€197 | Mini-Kurs → Strategiegespräch |
| **Apps / Tools** | Free Trial / Freemium | €0 | Lead-Magnet → App-Download |

### 6.3 Termin-Qualifizierung

Vor dem Termin wird geprüft:
- Alter (ideal 35+)
- Problem/Bedarf klar?
- Budget vorhanden?
- Entscheidungsbefugnis?

**Tool:** Typeform-Fragebogen vor Calendly-Buchung.

## 10. KPIs pro Funnel-Stufe

| Stufe | KPI | Ziel |
|-------|-----|------|
| Traffic | Cost per View (CPV), Cost per Click (CPC) | CPV <€0,03, CPC <€1 |
| Lead-Magnet | Conversion Rate Landing Page | >30 % |
| E-Mail | Open Rate, Click Rate | OR >35 %, CR >5 % |
| SMS/WhatsApp | Open Rate, Antwortrate | OR >80 %, Antwort >15 % |
| Sale | Conversion Lead → Kunde | >5 % |
| Termin | Buchungsrate, Show-Up-Rate | Buchung >10 %, Show-Up >70 % |
| High-Ticket | Conversion Termin → Sale | >20 % |
| CLV | Kundenwert über 12 Monate | >€150 |

## 11. Funnel-Optimierung: Test-Roadmap

| Woche | Test | Erwartung |
|-------|------|-----------|
| 1–2 | 3 verschiedene Hooks im Video | Höhere 3-Sekunden-View-Rate |
| 3–4 | 2 Landing-Page-Varianten | Höhere Lead-Conversion |
| 5–6 | 2 E-Mail-Betreffzeilen | Höhere Open Rate |
| 7–8 | Angebotspreis €7 vs. €0 | Bessere Lead-Qualität |
| 9–10 | SMS vs. E-Mail allein | Höhere Conversion |
| 11–12 | Bundle vs. Einzelprodukt | Höherer AOV |

---

*Funnel-Design basiert auf bewährten E-Commerce- und Lead-Generation-Prinzipien. Jeder Funnel muss mit echten Daten getestet und iteriert werden.*
