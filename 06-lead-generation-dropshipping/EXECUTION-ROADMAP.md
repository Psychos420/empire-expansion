# Execution Roadmap: 30/60/90 Tage

## Prinzip

> **In 90 Tagen von der Idee zu einem validierten, automatisierten Dropshipping-Funnel mit messbaren ersten Umsätzen.**

Jeder Meilenstein enthält konkrete Aufgaben, KPIs und die Verantwortlichkeit (Mensch vs. Agent/KI).

---

## Verantwortlichkeiten

| Rolle | Aufgaben |
|---|---|
| **Mensch (Du / Operative Leitung)** | Strategie, Lieferantenauswahl, Compliance, Budget, Kreative Richtung, Eskalationen |
| **Agent / KI (Kimi/Claude)** | Copywriting, Recherche, Berichte, E-Mail-Entwürfe, Analyse, Chatbot-Training |
| **n8n / Make / Zapier** | Workflow-Automation zwischen Shopify, E-Mail, SMS, Lieferanten |
| **Freelancer / VA (optional)** | Design, Videobearbeitung, Kundenservice Level 1 |

---

## Phase 1: Tage 1–30 – Setup & erste Validierung

### Ziel
Grundgerüst steht: Nische gewählt, Shop live, Lead-Magnet fertig, erster Traffic-Test läuft.

### Meilensteine & Aufgaben

| # | Aufgabe | Verantwortlich | Fertig bis |
|---|---|---|---|
| 1.1 | Nische finalisieren (Empfehlung: Schlaf & Erholung) | Mensch | Tag 3 |
| 1.2 | Markenname, Domain, Logo, Farbwelt definieren | Mensch + Agent | Tag 5 |
| 1.3 | Shopify-Store einrichten (Impressum, AGB, Datenschutz, Zahlungen) | Mensch + Agent | Tag 10 |
| 1.4 | 3–5 EU-Lieferanten kontaktieren & bewerten (Spocket, BigBuy, lokale Großhändler) | Mensch | Tag 12 |
| 1.5 | 10–15 Produkte importieren, Produktseiten optimieren | Agent + Mensch | Tag 15 |
| 1.6 | Lead-Magnet erstellen (PDF + Landing Page) | Agent + Mensch | Tag 18 |
| 1.7 | E-Mail-Tool (Klaviyo/Brevo) einrichten, Willkommens-Sequenz programmieren | n8n/Tool + Mensch | Tag 22 |
| 1.8 | Meta Pixel, GA4, UTM-Tracking implementieren | Mensch + Agent | Tag 24 |
| 1.9 | Erste Meta-Ads-Kampagne mit EUR 300–500 Budget starten | Mensch | Tag 25 |
| 1.10 | Erste 100 Leads sammeln & analysieren | Tool + Mensch | Tag 30 |

### KPIs Phase 1

| KPI | Ziel |
|---|---|
| Shop live | Ja |
| Produkte online | 10–15 |
| Leads generiert | ≥ 100 |
| Cost per Lead (CPL) | < EUR 5 |
| Landing-Page-Conversion | > 20 % |
| Erste Verkäufe | ≥ 3 |

### Risiken & Gegenmaßnahmen

| Risiko | Gegenmaßnahme |
|---|---|
| Lieferant findet sich nicht | 5 Alternativen parallel kontaktieren, Fokus auf EU |
| Lead-Magnet floppt | 2 Varianten testen, Hook optimieren |
| Shop-Compliance fehlt | E-Recht24 oder Anwalt prüfen lassen |

## Phase 2: Tage 31–60 – Optimierung & Skalierung

### Ziel
Funnel funktioniert, Traffic-Kanäle diversifiziert, erste Automatisierungen laufen, Umsatz steigt.

### Meilensteine & Aufgaben

| # | Aufgabe | Verantwortlich | Fertig bis |

| 2.1 | Analyse Phase 1: Beste Creatives, beste Produkte, beste Zielgruppe | Agent + Mensch | Tag 35 |
| 2.2 | Landing Page A/B-Testen (Headline, CTA, Formularfelder) | Agent + Tool | Tag 38 |
| 2.3 | E-Mail-Sequenz erweitern (7-Tage + Nurturing + Käufer-Sequenz) | Agent + Tool | Tag 42 |
| 2.4 | Google Ads / Google Shopping-Kampagne starten | Mensch | Tag 45 |
| 2.5 | TikTok-Konto & erste 5–10 Videos (UGC-Stil) veröffentlichen | Agent + Mensch | Tag 48 |
| 2.6 | WhatsApp/SMS-Channel aufsetzen für Warenkorb-Abbrüche | n8n/Tool | Tag 50 |
| 2.7 | Warenkorb-Abbruch-Workflow automatisieren | n8n/Tool | Tag 52 |
| 2.8 | Review-Request-Workflow automatisieren | Tool | Tag 55 |
| 2.9 | Bundle- und Abo-Angebote einführen | Mensch | Tag 58 |
| 2.10 | Erste 500 Leads, Umsatzziel EUR 1.000–2.000 | Team | Tag 60 |

### KPIs Phase 2

| KPI | Ziel |
|---|---|
| Leads gesamt | ≥ 500 |
| CPL | < EUR 4 |
| E-Mail-Öffnungsrate | > 25 % |
| Lead-to-Customer-Rate | > 4 % |
| Umsatz (brutto) | EUR 1.000–2.000 |
| ROAS (Paid) | > 2,0 |
| Retourenquote | < 15 % |

### Risiken & Gegenmaßnahmen

| Risiko | Gegenmaßnahme |
|---|---|
| ROAS zu niedrig | Creatives verbessern, Zielgruppe verfeinern, SEO ausbauen |
| E-Mail-Öffnungsrate niedrig | Betreffzeilen testen, Segmentierung verbessern |
| TikTok floppt | Fokus auf Meta + Google, TikTok nur testweise |

| 3.3 | Reporting-Dashboard in Looker Studio aufbauen | Agent + Tool | Tag 75 |
| 3.4 | Affiliate-/Influencer-Programm starten (5–10 Partner) | Mensch | Tag 78 |
| 3.5 | Retention-Strategie: Kundenclub, VIP-Status, Referral-Programm | Mensch + Agent | Tag 82 |

| 3.9 | Umsatzziel Phase 3: EUR 3.000–5.000 | Team | Tag 90 |

### KPIs Phase 3

| KPI | Ziel |
|---|---|

| CPL | < EUR 3 |
| ROAS (Paid) | > 3,0 |

| Phase | Fokus | Haupt-KPI | Budget (geschätzt) |
|---|---|---|---|

| Tage 31–60 | Optimierung & Skalierung | 500 Leads, ROAS > 2 | EUR 2.000–3.000 |
| Tage 61–90 | Automation & Expansion | 2.000 Leads, ROAS > 3 | EUR 2.500–4.000 |

**Gesamtbudget 90 Tage:** ca. EUR 6.000–9.000 (exkl. Tool-Kosten)

## Verantwortungsmatrix (RACI)

| Aufgabe | Mensch | Agent/KI | Tool/Automation |
|---|---|---|---|
| Strategie & Nischenwahl | R | C | I |
| Shop-Setup & Compliance | R | C | I |
| Lieferantenakquise | R | C | I |
| Produktlistings & Copy | A | R | C |
| Lead-Magnet & Content | A | R | C |
| Ads-Setup & Budget | R | C | I |
| E-Mail/SMS-Automation | A | C | R |
| Kundenservice Level 1 | C | R | R |
| Reporting & Analyse | A | R | R |
| Skalierung & Expansion | R | C | I |

**Legende:** R = Responsible (ausführend), A = Accountable (verantwortlich), C = Consulted (beratend), I = Informed (informiert)

---

## Wöchentliche Rhythmen

| Rhythm | Inhalt | Verantwortlich |
|---|---|---|
| **Täglich (10 Min.)** | Umsatz, Traffic, Ads-Spend prüfen | Mensch |
| **Wöchentlich (60 Min.)** | Lead-Kosten, Conversion, Top-Produkte, Retouren reviewen | Mensch + Agent |
| **Wöchentlich (90 Min.)** | Content-Kalender, neue Creatives planen | Agent + Mensch |
| **Monatlich (2–3 Std.)** | Strategie-Review, Budget-Shift, Nischen-Evaluation | Mensch + Agent |

---

## Erfolgskriterien nach 90 Tagen

- [ ] Shop läuft stabil und compliant.
- [ ] Mindestens 2.000 Leads in der Datenbank.
- [ ] Funnel ist zu > 70 % automatisiert.
- [ ] Erste wiederkehrende Umsätze durch Abo/Retention.
- [ ] ROAS > 3,0 auf mindestens einem Paid-Kanal.
- [ ] Entscheidung über Skalierung oder zweite Nische ist datenbasiert getroffen.

---

## Nächste Schritte (sofort)

1. Nische „Schlaf & Erholung“ bestätigen.
2. Domain & Shopify-Account sichern.
3. 5 EU-Lieferanten recherchieren und kontaktieren.
4. Lead-Magnet-Thema finalisieren („7-Tage-Sleep-Plan“).
5. Erste EUR 500 Ad-Budget freigeben.

---

## Fazit

Die Roadmap ist aggressiv, aber realistisch. Der Fokus liegt auf **schneller Validierung, automatisierbarer Skalierung und Lead-First-Denken**. Nach 90 Tagen liegt nicht nur ein funktionierender Shop vor, sondern ein **wiederverkäufliches Kunden-Ökosystem**, das die Basis für weitere Empire-Expansion-Bereiche bildet.
