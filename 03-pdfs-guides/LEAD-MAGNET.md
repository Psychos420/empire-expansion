## Kapitel 1: Warum die meisten Deutschen unterschätzen, was sie wirklich brauchen

### Die drei größten Renten-Fehler ab 40

| Fehler | Warum er passiert | Was es kostet |
|--------|-------------------|---------------|
| **1. „Die gesetzliche Rente reicht schon"** | Wir unterschätzen steigende Lebenserhaltungskosten und Inflation. | Tausende Euro pro Jahr Rentenlücke |
| **2. „Ich fange nächstes Jahr an"** | Aufschub kostet Zinseszins. | Bei 5 % Rendite kostet jedes Jahr Verzögerung ca. 5–8 % Endvermögen |
| **3. „Ich kaufe, was mein Berater empfiehlt"** | Hohe Gebühren, unpassende Produkte, mangelnde Transparenz. | 1–2 % Gebühren pro Jahr fressen langfristig ein Drittel des Vermögens |

### Die wichtigste Zahl: Ihr Rentenbedarf

Viele Menschen rechnen mit 70–80 % ihres letzten Nettoeinkommens als Rentenbedarf. In der Praxis reicht das oft nicht, weil:

- Kranken- und Pflegeversicherung weiter anfallen,
- Miete und Nebenkosten steigen,
- Freizeit, Reisen und Enkelkinder Geld kosten,
- unvorhergesehene Reparaturen und Gesundheitskosten anfallen.

**Faustformel für Ihren Rentenbedarf:**

```
Rentenbedarf pro Monat = Aktuelle Nettoausgaben × 1,1
```

Das ist keine wissenschaftliche Formel, aber ein realistischer Startpunkt. Notieren Sie Ihre Zahl:

**Mein geschätzter Rentenbedarf pro Monat: __________ €**

## Kapitel 2: Checkliste 1 — Ihre gesetzliche Rente realistisch einschätzen

### Schritt 1: Renteninformation finden

Jeder gesetzlich Versicherte erhält jährlich eine **Renteninformation** von der Deutschen Rentenversicherung (bzw. dem zuständigen Träger in AT/CH). Suchen Sie das letzte Schreiben oder loggen Sie sich online ein unter:

- **Deutschland:** [www.deutsche-rentenversicherung.de](https://www.deutsche-rentenversicherung.de)
- **Österreich:** [www.pensionsversicherung.at](https://www.pensionsversicherung.at)
- **Schweiz:** AHV-Ausgleichskasse bzw. [www.ahv-iv.ch](https://www.ahv-iv.ch)

### Schritt 2: Entscheidende Zahlen herauslesen

Tragen Sie ein:

| Punkt | Wert |
|-------|------|
| Geschätzte monatliche Rente bei Regelaltersrente | __________ € brutto |
| Geschätzte monatliche Rente nach Steuern (ca. 80–90 % brutto) | __________ € netto |
| Regelaltersrente / aktuelles Renteneintrittsalter | __________ Jahre |
| Anzahl bisheriger Beitragsjahre | __________ Jahre |

### Schritt 3: Realitäts-Check

Die Renteninformation zeigt oft das **Best-Case-Szenario** bei gleichbleibendem Einkommen bis zur Rente. Beantworten Sie ehrlich:

- [ ] Ich werde bis zur Rente ununterbrochen gleich viel verdienen.
- [ ] Ich werde keine längeren Arbeitslosigkeits- oder Erwerbsminderungsphasen haben.
- [ ] Die gesetzliche Rente wird nicht weiter gekürzt oder Renteneintrittsalter nicht erhöht.
- [ ] Mein Berufsstand wird weiterhin gleichbleibend Beiträge zahlen.

Wenn Sie nicht alle vier Kästchen ankreuzen können, gehen Sie von einem Abschlag von **10–20 %** auf die geschätzte Rente aus.

**Meine realistische monatliche gesetzliche Rente (netto): __________ €**

## Kapitel 3: Checkliste 2 — Die Rentenlücke in 10 Minuten berechnen

### Die Formel

```
Rentenlücke pro Monat = Rentenbedarf − realistische gesetzliche Rente
Rentenlücke pro Jahr  = Rentenlücke pro Monat × 12
```

### Beispielrechnung

| Position | Betrag |
|----------|--------|
| Geschätzter monatlicher Rentenbedarf | 3.200 € |
| Realistische gesetzliche Rente (netto) | 1.500 € |
| **Monatliche Rentenlücke** | **1.700 €** |
| **Jährliche Rentenlücke** | **20.400 €** |

### Ihre Rechnung

| Position | Betrag |
|----------|--------|
| Mein monatlicher Rentenbedarf | __________ € |
| Meine realistische gesetzliche Rente (netto) | __________ € |
| **Meine monatliche Rentenlücke** | **__________ €** |
| **Meine jährliche Rentenlücke** | **__________ €** |

### Was bedeutet die Lücke?

| Jährliche Lücke | Handlungsbedarf |
|-----------------|-----------------|
| 0–6.000 € | Gering. ETF-Sparplan mit 100–250 €/Monat prüfen. |
| 6.000–15.000 € | Mittel. ETF-Sparplan 250–500 €/Monat plus Steuervorteile nutzen. |
| 15.000–30.000 € | Hoch. Sparplan 500–1.000 €/Monat, möglicherweise Zusatzrente, Vermietung o. ä. |
| 30.000+ € | Sehr hoch. Sofortiger Aktionsplan: Einkommen steigern, Ausgaben senken, professionelle Beratung. |

**Mein Handlungsbedarf: __________**

## Kapitel 4: Checkliste 3 — Notgroschen, Schulden & Sicherheit vor dem Sparen

### Bevor Sie anlegen: 3 Sicherheitsfragen

Beantworten Sie diese Fragen, bevor Sie Geld in ETFs oder andere Anlagen stecken.

#### 1. Haben Sie einen Notgroschen?

Empfohlene Höhe: **3–6 Monatsausgaben** auf einem Tagesgeld- oder Girokonto.

- [ ] Ja, vorhanden: __________ €
- [ ] Nein, noch im Aufbau. Ziel: __________ €

**Regel:** Bauen Sie den Notgroschen zuerst auf, bevor Sie größere Summen anlegen.

#### 2. Haben Sie teure Schulden?

Teure Schulden (Dispo, Kreditkarte, Ratenkredite mit > 6 % Zinsen) sollten vor einer Anlage zurückgeführt werden.

| Schuldenart | Restschuld | Zinssatz |
|-------------|------------|----------|
| Dispo/Kreditkarte | __________ € | __________ % |
| Ratenkredit | __________ € | __________ % |
| Autokredit | __________ € | __________ % |
| Sonstiges | __________ € | __________ % |

- [ ] Ich habe keine teuren Schulden.
- [ ] Ich werde teure Schulden zuerst tilgen, bevor ich neu anlege.

#### 3. Sind Sie ausreichend abgesichert?

- [ ] Krankenversicherung: Vorhanden und leistungsfähig
- [ ] Berufsunfähigkeitsversicherung: Vorhanden oder geprüft
- [ ] Haftpflichtversicherung: Vorhanden
- [ ] Sach- und Wohngebäudeversicherung: Vorhanden (falls Eigentum)

> **Hinweis:** Eine Berufsunfähigkeitsversicherung ist für 40+ oft wichtiger als ein weiterer ETF-Sparplan. Ohne Einkommen können Sie nicht weiter sparen.

|-------|--------------------------|----------------|
| 40–45 | 70–80 % | 20+ Jahre |
| 46–50 | 60–70 % | 15–20 Jahre |
| 51–55 | 50–60 % | 10–15 Jahre |
| 56–60 | 40–50 % | 7–10 Jahre |
| 60+ | 30–40 % | 5–7 Jahre |

### Die einfachste ETF-Strategie für Einsteiger

**Core-Satellite-Strategie:**

| Position | Produkttyp | Anteil | Zweck |
|----------|------------|--------|-------|
| Core | Welt-ETF (thesaurierend) | 60–80 % | Breite Streuung, langfristiges Wachstum |
| Satellite 1 | Europa- oder DAX-ETF | 10–20 % | Heimatmarkt, Währungseffekte |
| Satellite 2 | Anleihen-ETF oder Tagesgeld | 10–30 % | Stabilität, Abschwungpuffer |

> **Tipp:** Für die meisten Anleger 40+ reicht ein einziger thesaurierender Welt-ETF als Kern. Details dazu finden Sie im kostengünstigen Folgeprodukt „ETF-Strategie für 40+".
