Formularübermittlung technisch über Web3Forms (web3forms.com). Die eingegebenen Daten werden dabei an Web3Forms-Server übertragen und per E-Mail an uns weitergeleitet.

Für die Buchung eines Erstgesprächs nutzen wir Cal.com. Bei einer Terminbuchung werden die von dir angegebenen Daten (Name, E-Mail, gewünschter Termin) an Cal.com übermittelt und zur Terminorganisation verwendet.

Hosting

Diese Website wird über GitHub Pages (GitHub Inc. / Microsoft) gehostet. Beim Aufruf der Seite werden technisch bedingt Server-Logs (IP-Adresse, Zeitpunkt, aufgerufene Seite) durch GitHub verarbeitet.

Schriftarten

Wir nutzen Google Fonts zur Darstellung der Schriftarten dieser Seite. Beim Laden der Seite kann dadurch eine Verbindung zu Servern von Google hergestellt werden.
