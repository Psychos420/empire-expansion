# Execution-Roadmap: 30/60/90-Tage-Plan

## Grundsätzliche Rollenverteilung

| Rolle | Verantwortlichkeit | Mensch vs. Agent |
|-------|---------------------|------------------|
| **Strategie & Positionierung** | Markenstimme, USP, Preise, Kanäle | Mensch |
| **Content-Erstellung** | Posts, Newsletter, Scripts, Lead-Magnete | Agent erstellt Entwürfe, Mensch prüft |
| **Technische Umsetzung** | Landing Pages, Funnel, Automatisierungen | Mensch + Agent zusammen |
| **Lead-Generierung** | Ads, SEO, Outreach, Webinare | Mensch steuert, Agent unterstützt |
| **Verkauf** | Gespräche, Angebote, Abschlüsse | Mensch |

| **Reporting & Optimierung** | KPIs, A/B-Tests, Reporting | Agent sammelt Daten, Mensch entscheidet |

## Verantwortlichkeiten: Mensch vs. Agent

### Mensch (Unternehmer / Strategischer Kopf)

| Aufgabe | Warum menschlich? |
|---------|-------------------|
| Positionierung und Preisgestaltung | Strategische Entscheidung |
| Verkaufsgespräche | Vertrauensaufbau, Verhandlung |
| Kundenbeziehung & Support | Persönlicher Kontakt |
| Strategic Partnerships | Netzwerk, Verhandlung |
| Content-Review & Freigabe | Markenstimme, Qualität |
| Angebotsstellung & Verträge | Rechtliche Verantwortung |
| Investitionsentscheidungen | Budget, Risiko |

### Agent (KI / Automatisierung)

| Aufgabe | Warum Agent? |
|---------|--------------|
| Content-Entwürfe | Geschwindigkeit, Skalierbarkeit |
| E-Mail-Sequenzen | Wiederholbare Templates |
| Landing-Page-Texte | A/B-Test-Variationen |
| Datenrecherche & Reporting | Strukturierte Auswertung |
| Code-Snippets für n8n/Make | Technische Assistenz |
| Social-Media-Posts | Hohe Frequenz |
| Dokumentation & SOPs | Konsistenz |

### Hybride Zusammenarbeit

| Aufgabe | Modus |
|---------|-------|
| Funnel-Architektur | Agent entwirft, Mensch entscheidet |
| Ad-Kampagnen | Mensch strategisch, Agent erstellt Texte/Keywords |
| Kundenprojekte | Mensch leitet, Agent assistiert bei Prompts/Workflows |
| A/B-Tests | Agent analysiert Daten, Mensch interpretiert |

## Risiken & Contingency-Plan

| Risiko | Wahrscheinlichkeit | Auswirkung | Maßnahme |
|--------|-------------------|------------|----------|
| Zu wenige Leads | Mittel | Hoch | Budget auf Google Ads erhöhen, mehr Outreach |
| Leads konvertieren nicht | Mittel | Hoch | Landing Page optimieren, bessere Lead-Qualifikation |
| Technische Probleme bei Kunden | Niedrig | Hoch | Monitoring, Backup-Pläne, klare SLAs |
| Wettbewerber senken Preise | Mittel | Mittel | Auf Wert und Spezialisierung bestehen |
| Kunde erwartet zu viel | Hoch | Mittel | Klare Scopes, realistische Erwartungen |
| Zeitmangel beim Umsetzer | Hoch | Hoch | Fokus auf MVP, Freelancer für Spitzen |
